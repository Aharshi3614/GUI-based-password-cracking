🔐 GUI-Based Password Cracking Tool

A Python-based application with a graphical user interface (GUI) that demonstrates password cracking techniques such as brute-force and dictionary attacks. This project is built for educational and cybersecurity awareness purposes only.

📌 Features

🖥️ User-friendly GUI using Tkinter

🔑 Supports:

Brute-force attack

Dictionary attack

⚡ Real-time output display

📊 Shows:

Attempted passwords

Time taken

Success/failure result


🚀 Technologies Used

Python 3

Tkinter (for GUI)

Hashlib (for password hashing)

Time module (for performance tracking)
